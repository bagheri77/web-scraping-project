وب‌اسکرپینگ و اتوماسیون داده‌ها با استفاده از Selenium

این پروژه برای استخراج داده‌های تغذیه‌ای از وب‌سایت‌ها و ذخیره‌سازی آن‌ها در فرمت‌های Excel و JSON طراحی شده است. از Selenium برای اتوماسیون مرورگر و pandas برای پردازش داده‌ها استفاده می‌شود.

ویژگی‌ها :

استخراج خودکار داده‌ها از صفحات وب.

ذخیره‌سازی داده‌ها به فرمت‌های Excel و JSON.

استفاده از Selenium برای کنترل مرورگر و requests برای ارسال درخواست‌های HTTP.

پردازش داده‌ها با استفاده از pandas و انجام محاسبات با numpy.

پیش‌نیازها :

قبل از اجرای پروژه، اطمینان حاصل کنید که موارد زیر نصب شده است:

پایتون نسخه 3.10 یا بالاتر

ChromeDriver (برای Selenium) و مرورگر کروم
نکته : حتما باید نسخه های مرورگر کروم و ChromeDriver با هم سازگار باشند .

نصب وابستگی‌ها :

نصب محیط مجازی (اختیاری ولی توصیه‌شده):
برای استفاده از یک محیط ایزوله و مستقل از سایر پروژه‌ها، محیط مجازی را ایجاد کنید:

python -m venv myenv


فعال‌سازی محیط مجازی:

برای ویندوز:

myenv\Scripts\activate

نصب وابستگی‌ها:
پس از فعال‌سازی محیط مجازی، وابستگی‌ها را از فایل requirements.txt نصب کنید:

pip install -r requirements.txt


اجرای پروژه:

پس از نصب وابستگی‌ها و تنظیم ChromeDriver، پروژه را با اجرای کد زیر آغاز کنید:

python final.py


عملیات کد:

این پروژه به طور خودکار از سایت https://badankhooba.com/calculating-calories/ داده‌های تغذیه‌ای را استخراج کرده و در فایل‌های Excel و JSON ذخیره می‌کند.
